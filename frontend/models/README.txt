How to export from Mixamo:
1. Go to mixamo.com → pick character + idle animation
2. Download as FBX (With Skin)
3. Convert FBX → GLB at: https://products.aspose.app/3d/conversion/fbx-to-gltf
4. Rename the output file to match the names above
5. Drop it in this folder
6. Refresh the Characters page — the 3D model replaces the flat image automatically
